NBA ID Sign-In Beta Description

Joe Chindemi - IDM-241-001 Microinteractions

Click to see the NBA.com sign-in

TRIGGERS

  • Hovering on/hovering off the "Sign In" button.
  • Clicking on the "Sign In" button.

RULES

HOVER

  • ON: Button adopts a white background with black text and a black border.
  • ON: Cursor changes to a hand pointer.
  • OFF: Button changes back to it's original black background with white text while keeping the black border.
  • OFF: Cursor changes back to the standard pointer arrow.

CLICK

  • Logo div on right side will fade to a GIF of a basketball being scored to signify a succesful sign-in.
  • Sign-in information is being processed

FEEDBACK

HOVER

  • ON: Button adopts a white background with black text and a black border.
  • ON: Cursor changes to a hand pointer.
  • OFF: Button changes back to it's original black background with white text while keeping the black border.
  • OFF: Cursor changes back to the standard pointer arrow.

CLICK

  • Logo div on right side will fade to a GIF of a basketball being scored to signify a succesful sign-in.

LOOPS/MODES

  • Loop: The GIF only plays once after signing in.
